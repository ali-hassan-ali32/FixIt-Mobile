# Fix It – Handyman Service Platform

## Overview

Fix It is a platform designed to connect customers with skilled handymen for maintenance, repair, and home services. The platform simplifies the process of finding trusted professionals, requesting services, and managing service requests through a modern and user-friendly experience.

This project was developed as a Graduation Project for the Computer Science Department at Higher Technological Institute (HTI).

---

## Problem Statement

Finding reliable handymen for maintenance and repair tasks is often difficult and time-consuming. Customers struggle to locate qualified professionals, compare services, and manage service requests efficiently.

Fix It addresses this problem by providing a centralized platform that connects customers with verified handymen across multiple service categories.

---

## Features

### Customer Features

* User Registration & Login
* Profile Management
* Browse Service Categories
* View Handyman Profiles
* View Portfolio and Previous Work
* Create Service Requests
* Track Request Status
* Rate and Review Handymen

### Handyman Features

* Handyman Registration
* Profile Management
* Portfolio Management
* Receive Service Requests
* Update Job Status
* View Customer Reviews

### Admin Features

* Manage Users
* Manage Handymen
* Approve Handyman Accounts
* Manage Categories
* Monitor Platform Activity
